# Cyano Theme for VS Code

![dark](https://raw.githubusercontent.com/diego1araujo/cyano-theme/images/cyano-dark.png)

### Cyano Dark Blue

![dark-blue](https://raw.githubusercontent.com/diego1araujo/cyano-theme/images/cyano-dark-blue.png)

## Installation

- Open `Extensions` in VS Code. Go to `View` -> `Extensions` in **Menu Bar**
- Search for `Cyano`
- Click on `Install` button to install it

## Activating the Theme

- Open `Command Palette` in VS Code. Go to `View` -> `Command Palette` in **Menu Bar**
- Search for `Color Theme` and select it
- Search for `Cyano` and choose either `Cyano Dark` or `Cyano Dark Blue`

### Icon Theme

The file icon theme in the screenshot is [Ayu Theme](https://github.com/ayu-theme/vscode-ayu)

*Workbench color & Syntax color based on [Ayu Theme](https://github.com/ayu-theme/vscode-ayu) and [Panda Syntax Theme](https://github.com/tinkertrain/panda-syntax-vscode)*
